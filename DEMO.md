# 데모 가이드 (발표용)

## 실행
```powershell
.\run_demo.ps1
```
→ 서버가 뜨고 브라우저로 http://127.0.0.1:8000/ 가 열립니다. (종료: Ctrl+C)

수동 실행이 필요하면:
```powershell
py -3 -m uvicorn service.app:app --port 8000
```

## 시연 흐름 (4단계, 약 2분)

1. **검색** — 상단 검색창에 `smart farm` 입력 → [검색]. 796장 레퍼런스에서 결과가 그리드로 뜸.
   - 필터 [다크]/[표지] 토글로 좁히기. 색으로도 검색 가능: 색 칸에 `#15402b`.
   - 초록색 **"레이아웃" 배지**가 붙은 카드 = 실제 레이아웃 복사가 가능한 레퍼런스.

2. **선택** — "레이아웃" 배지가 있는 카드를 클릭. 오른쪽 패널에
   "이 레퍼런스의 실제 레이아웃을 복사해 채웁니다" 표시, 테마가 자동 적용됨.

3. **자동 채움 → 생성** — 오른쪽에서 제목/부제/체크리스트를 원하는 문구로 바꾸고
   (`*키워드*` 는 강조 색) [생성] 클릭 → 미리보기에 카드가 렌더됨.
   - 배지 있는 레퍼런스 = "레이아웃 복사", 없으면 "테마 폴백"으로 표시.

4. **(선택) 덱 내보내기** — `POST /api/deck` 로 표지+내지 여러 장을 ZIP으로.

## 말할 포인트
- ML은 **분석(검출기)+검색(레퍼런스 라이브러리)** 에 쓰고, 픽셀은 **결정적 렌더링**.
  GAN으로 한글을 그리지 않음 → "AI같음" 탈피, 디자이너급 품질.
- 엔진은 손으로 짠 한 가지 스타일이 아니라 **레퍼런스마다 추출한 레이아웃 템플릿**을 복사.
- 웹 UI는 **WCAG 2.1 AA** 통과(키보드만으로 검색→선택→생성 가능).

- `service/out/deck_00.png`, `deck_01.png` — 엔진 v2 품질
- `gen_output/refill/refill_list_img_001.png` — 원본 | 리필 비교

```powershell
py -3 reflib\search.py --text "smart farm" --dark --k 8
py -3 gen\refill_demo.py        # 레이아웃 복사 데모 -> gen_output\refill\
```
