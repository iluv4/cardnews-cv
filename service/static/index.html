카드뉴스 스튜디오

① 기획

누구에게, 무엇을 전할지 먼저 정합니다. 여기서 정한 키워드는 ③ 스타일 검색에 자동으로 쓰입니다.

내부 관리용 이름이에요. 카드에는 표시되지 않습니다.

카드 우상단에 들어가는 브랜드 마크. 비우면 표시 안 함.

레퍼런스를 고를 때 쓰는 영문/주제 키워드.

② 구성

카드뉴스는 여러 장이 한 편입니다. 커버 → 본문 → 마무리 순서로 카드를 구성하세요. 종류는 입력한 내용으로 자동 판별됩니다(체크리스트가 있으면 체크리스트, 부제만 있으면 본문, 첫 장은 커버).

③ 스타일

실제 카드뉴스 레퍼런스를 검색해 하나를 고르면, 그 톤·팔레트가 덱 전체에 적용됩니다. 레이아웃 배지가 있는 레퍼런스는 실제 레이아웃까지 그대로 복사합니다.

선택된 레퍼런스가 없습니다 — 선택 없이 진행하면 기본 테마로 생성됩니다.

④ 생성 · 검토

덱 전체를 렌더링합니다. 카드를 클릭하면 크게 볼 수 있고, ‘수정’을 누르면 구성 단계로 돌아갑니다.

‘덱 생성’을 누르면 카드들이 여기에 나타납니다.

⑤ 내보내기

완성된 덱을 ZIP으로 내려받고, 결과가 좋았는지 평가해 주세요. 평가는 다음 검색 순위에 반영됩니다.
